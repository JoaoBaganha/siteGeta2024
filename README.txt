
Grupo de Estudos em Tecnologia Assistiva do CESUPA (GETA CESUPA)

Bem-vindo ao repositório do site do Grupo de Estudos em Tecnologia Assistiva do Centro Universitário do Pará (CESUPA)!

Sobre o Projeto
Este projeto foi desenvolvido para criar o site do GETA CESUPA, um grupo de estudos dedicado ao desenvolvimento e pesquisa de soluções tecnológicas voltadas para pessoas com deficiência e em condições de vulnerabilidade.

Tecnologias Utilizadas
O site foi criado utilizando as seguintes tecnologias:

HTML: Utilizado para estruturar o conteúdo do site.
CSS: Utilizado para estilizar o conteúdo e proporcionar uma melhor experiência visual.
JavaScript: Utilizado para adicionar interatividade e funcionalidades dinâmicas ao site.
